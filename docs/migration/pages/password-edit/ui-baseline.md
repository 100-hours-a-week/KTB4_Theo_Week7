# 비밀번호 수정 텍스트 UI 기준선

## 1. 전체 배치

```text
┌──────────────────────────────────────────────────────────┐
│                     </> 코드 한입             [프로필]  │
├──────────────────────────────────────────────────────────┤
│                                                          │
│                     비밀번호 수정                        │
│            안전한 비밀번호로 계정을 보호해주세요.        │
│                                                          │
│        ┌──────────── 비밀번호 card ─────────────┐        │
│        │ 비밀번호                               │        │
│        │ 8~20자, 대·소문자·숫자·특수문자 안내  │        │
│        │ [비밀번호를 입력하세요               ] │        │
│        │ [비밀번호 helper]                     │        │
│        │                                        │        │
│        │ 비밀번호 확인                          │        │
│        │ 위에 입력한 비밀번호와 동일하게 안내   │        │
│        │ [비밀번호를 한번 더 입력하세요       ] │        │
│        │ [확인 helper]                          │        │
│        │                                        │        │
│        │              [수정하기]                │        │
│        └────────────────────────────────────────┘        │
│                                                          │
│                         (수정완료 toast)                  │
└──────────────────────────────────────────────────────────┘
```

## 2. Header와 본문

- Header 높이: 데스크톱 78px, 760px 이하 70px
- 중앙 로고와 우측 프로필 메뉴, 뒤로가기 없음
- main 상단 padding 58px
- form container 최대 너비 600px
- 제목 32px, 모바일 28px
- 설명은 보조색 15px

## 3. Form card

- 어두운 반투명 배경, 14px radius, blur와 shadow
- 데스크톱 padding 34px 38px 32px
- label은 14px 굵은 글자
- guide는 보조색 13px과 1.45 줄 간격
- input 높이 50px, focus 시 민트 border와 ring
- helper는 붉은색 12px, 최소 높이 18px
- 저장 버튼은 전체 너비·50px, 민트 gradient
- 비활성 버튼 opacity 0.55

## 4. 문구

| 위치 | 텍스트 |
|---|---|
| 비밀번호 guide | `8~20자, 대문자·소문자·숫자·특수문자를 각각 1개 이상 포함해주세요.` |
| 확인 guide | `위에 입력한 비밀번호와 동일하게 입력해주세요.` |
| 비밀번호 placeholder | `비밀번호를 입력하세요` |
| 확인 placeholder | `비밀번호를 한번 더 입력하세요` |
| submit | `수정하기` |
| toast | `수정완료` |

## 5. Toast

- 화면 하단 중앙 fixed pill
- `role="status"`
- 성공 후 2초 표시
- 회원정보 수정 toast와 동일한 시각 스타일

## 6. 반응형

### 760px 이하

- Header, main·card padding과 제목 크기 축소

### 520px 이하

- 로고를 20px, `</>`를 24px로 축소
