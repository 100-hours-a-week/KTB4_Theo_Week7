# 로그인 기준 화면 캡처

로그인 페이지의 Vanilla JS 기준 화면이 필요한 경우 이 디렉터리에 저장한다. 캡처는 선택 사항이며, 필수 화면 기준선은 `../ui-baseline.md`에 텍스트로 기록한다.

필요한 파일명과 촬영 조건은 `../checklist.md`의 **선택적 캡처 체크리스트**를 따른다.

## 주의사항

- 실제 비밀번호가 보이지 않는지 확인한다.
- 개발자 도구를 함께 캡처할 경우 Access Token과 쿠키 값을 가린다.
- 가능하면 모든 화면을 같은 브라우저 크기와 배율로 촬영한다.
