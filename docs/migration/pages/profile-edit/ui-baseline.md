# 회원정보 수정 텍스트 UI 기준선

## 1. 전체 배치

```text
┌──────────────────────────────────────────────────────────┐
│                     </> 코드 한입             [프로필]  │
├──────────────────────────────────────────────────────────┤
│                                                          │
│                    회원정보 수정                         │
│              프로필과 닉네임을 관리해보세요.             │
│                                                          │
│        ┌──────────── 프로필 card ───────────────┐        │
│        │              프로필 사진               │        │
│        │               (이미지)                 │        │
│        │             [이미지 변경]              │        │
│        │                                        │        │
│        │ 이메일                                 │        │
│        │ [user@example.com                    ] │        │
│        │ 이메일은 변경할 수 없습니다.           │        │
│        │                                        │        │
│        │ 닉네임                                 │        │
│        │ [기존 닉네임                         ] │        │

│        │ 띄어쓰기 없이 10자 이내 안내           │        │
│        │                                        │        │
│        │          [변경사항 저장]               │        │
│        └────────────────────────────────────────┘        │

│        │ 회원 탈퇴 시 모든 데이터가 삭제됩니다. │        │
```

GET 완료 전에는 Header, 제목과 설명만 보이며 form과 탈퇴 panel은 표시되지 않는다.

## 2. Header와 본문

- Header 높이: 데스크톱 78px, 760px 이하 70px
- 중앙 로고와 우측 프로필 메뉴, 뒤로가기 없음
- 본문 최대 너비 760px, form 최대 너비 600px
- 페이지 제목 32px, 모바일 28px
- 어두운 배경 이미지 위에 gradient 적용

## 3. 프로필 card

- 14px radius의 어두운 반투명 card, blur와 shadow
- 프로필 원형 132×132px, 모바일 120×120px
- 이미지가 없으면 원 안에 CSS 기본 아이콘 표시
- 원 아래 별도 `이미지 변경` outline control 표시
- 이메일은 흐린 읽기 전용 card 형태
- 닉네임 input은 focus 시 민트 border와 ring
- helper는 붉은색 12px, 최소 높이 18px
- 저장 버튼은 전체 너비 50px 높이, 민트 gradient
- 비활성 버튼 opacity 0.55

## 4. 회원 탈퇴

- form card 아래 별도 반투명 panel
- 붉은 계열 `회원 탈퇴 >` text button
- modal은 408px, 중앙 배치, 어두운 card와 backdrop blur
- modal 버튼은 취소·확인 순서이고 확인 버튼이 민트색
- 요청 중 두 버튼 모두 비활성화

## 5. Toast

- 화면 하단 중앙 fixed pill 형태
- 문구 `수정완료`
- `role="status"`
- 성공 직후부터 2초 동안 표시

## 6. 반응형

### 760px 이하

- Header, main·card padding과 프로필 원 크기 축소
- modal padding·버튼 너비 축소

### 520px 이하

- Header 로고를 20px, `</>`를 24px로 축소
